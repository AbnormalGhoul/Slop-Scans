# SLOP SCAN

## How to Run
```bash
./setup.sh
source venv/bin/activate
./start.sh
```
Server should start at:
```
http://localhost:8000
http://localhost:8000/docs
```

## Notes
- Models will download automatically on first run
- The first startup may take time due to model loading
- Backend was coded in Ubuntu (Windows startup commands will be different)
- Models: SightEngine API & Desklib API
